# 🎨 SpectraText

A simple project showcasing cool **CSS animations** for text!  

## 🚀 Features
- 🌊 Wavy Text Effect  
- ✨ Glowing Text Effect  
- ⌨️ Typing Effect  
- 🌈 Color Changing Effect  

## 📂 How to Use  
1. Clone this repo  
2. Open `index.html` in your browser  
3. Enjoy the animations! 🎉
